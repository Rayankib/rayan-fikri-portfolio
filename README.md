# Personal Portfolio — Rayan Fikri

A clean, minimal portfolio website showcasing selected creative projects, frontend development, and media creation work. This static site is built to present a professional portfolio experience without changing the visual design of the existing pages.

## Project overview

This portfolio presents:
- A concise hero introduction with personal branding.
- Selected project overviews with visual previews and case study pages.
- A short about section with skills and professional focus.
- Contact links for email, Instagram, and phone.

The portfolio is designed for internship applications, creative collaborations, and recruiter review.

## Target audience

- Internship recruiters and hiring managers
- Creative teams and designers
- Project collaborators and mentors
- Anyone reviewing frontend or media production work

## Features

- Responsive static site layout
- Custom typography using Google Fonts
- Project preview cards with image focus control
- Dedicated case pages for each project
- Smooth in-page navigation and clear contact section
- Simple hover interactions in JavaScript

## Technologies used

- HTML
- CSS
- JavaScript

## Repository structure

- `index.html` — Main portfolio homepage

- `.gitignore` — Local files excluded from Git

## Installation & local preview

No build tools are required. To preview locally:

1. Open the project folder in a code editor.
2. Open `index.html` in a browser.

Optional local server methods:

- Use VS Code Live Server and select `Open with Live Server`.
- Run a simple local server if available, for example with Python:
  ```bash
  python -m http.server 8000
  ```
  Then visit `http://localhost:8000`.
